# SQL Retail Sales Analysis Project

##  Overview
This project focuses on analyzing retail sales data using SQL.  
It includes database creation, data cleaning, exploration, and solving business-related queries to extract meaningful insights.

The goal of this project is to demonstrate SQL skills such as:
- Data querying
- Aggregation
- Filtering
- Window functions
- Business problem solving

---

##  Database Details
- **Database Name:** sql_project_retail
- **Table Name:** retail_sales

###  Table Columns:
- transactions_id (Primary Key)
- sale_date
- sale_time
- customer_id
- gender
- age
- category
- quantity
- price_per_unit
- cogs (Cost of Goods Sold)
- total_sale

---

##  Data Cleaning
Performed the following checks:
- Checked for NULL values in all columns
- Identified missing or inconsistent data
- Ensured data integrity before analysis

---

##  Data Exploration
- Total number of sales
- Unique customers count
- Unique product categories
- List of all categories

---

##  Business Problems Solved

### Key SQL Queries:
1. Sales on a specific date
2. Clothing sales with quantity > 4 in Nov 2022
3. Total sales per category
4. Average age of customers (Beauty category)
5. Transactions with sales > 1000
6. Transactions by gender and category
7. Best selling month in each year
8. Top 5 customers by total sales
9. Unique customers per category
10. Sales shift analysis (Morning, Afternoon, Evening)
